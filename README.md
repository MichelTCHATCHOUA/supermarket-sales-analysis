# 🛒 Supermarket Sales Analysis

Projet personnel de Data Science — Analyse exploratoire de données de ventes d’un supermarché.

---

## 🎯 Objectif

Analyser les ventes d’un supermarché à partir d’un jeu de données public.  
Extraire des insights utiles sur le chiffre d’affaires, les produits, les branches, les méthodes de paiement, les profils clients, etc.

---

## 📊 Données

- Source : [Kaggle – Supermarket Sales Dataset](https://www.kaggle.com/datasets/aungpyaeap/supermarket-sales)
- Format : CSV
- Contenu : 
  - Date et heure d’achat
  - Produit, catégorie, prix unitaire
  - Mode de paiement, évaluation client
  - Villes, branches, etc.

---

## 🛠️ Outils utilisés

- Python 3
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook
